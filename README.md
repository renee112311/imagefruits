# imagefruits
這是一個靜態影像創作的交流展示平台。  
[DEMO](https://renee112311.github.io/imagefruits/#/)
## 前台

### 前台開發工具
* Vue cli
* axios
* bootstrap-vue
* 網頁放在 github pages
* 使用 github actions 部屬
* 使用 Travis CI 自動打包
